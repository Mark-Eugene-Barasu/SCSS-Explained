SCSS Function:

Function

Functions can be values too! You can directly write a function as a value, but you can pass function's name to the [meta.get-function() function][]to get it as a value. Once you have a function value, you can pass it to the meta.exists() function to call it. This is useful for writing higher-order function to call another function.
